# Задание на разработку (Intake)

**Проект:** Информационная система «Академия Развития Человека»

**Цель:** Создание комплексного решения для управления расписанием, участниками и аналитикой академии с удобным мобильным интерфейсом для администраторов.

**Основные требования:**
1. **Расписание:**
   - Образец: текстовое расписание на апрель 2026.
   - Адаптивность: ПК, Мобильные устройства, Mini-app.
   - Горизонт планирования: 1 год.
   - Фильтрация: по месяцам или за год.
   - Скрытие прошедших событий (с возможностью просмотра архива).
   - Запись на события с отображением количества участников.
   - Верификация при входе для записи.

2. **Администрирование:**
   - Приоритет на мобильный интерфейс.
   - Голосовой ввод данных.
   - Учет участников: филиал, посещаемость.
   - Финансовый блок: оплаты участников, расходы на организацию.
   - Статистика: посещаемость, сравнение с предыдущими периодами.

3. **Ролевая модель:**
   - **Участник:** статус, статистика посещений, наставник, реферер.
   - **Руководитель:** доступ к статистике своего уровня.
   - **Спикер:** фильтр своих мероприятий, статистика по ним.

4. **Логика и проверки:**
   - Контроль конфликтов спикера (один спикер в разных местах одновременно).
   - Контроль конфликтов помещений (разные мероприятия в одном зале).
   - Напоминания о параллельных мероприятиях в разных локациях.

5. **Реферальная программа:**
   - Бонусы за привлечение новых участников.

6. **Дизайн:**
   - Соответствие цветовой гамме логотипа (радужный цветок: фиолетовый, синий, зеленый, желтый, оранжевый, красный).
